# Weather Data Analysis

## Project Overview

This project analyzes weather data using Python.

It performs:

- Data Loading
- Data Cleaning
- Data Analysis
- Visualization
- Report Generation

## Technologies

- Python
- Pandas
- Matplotlib

## Folder Structure

data/

visualizations/

report/

main.py

README.md

requirements.txt

## Charts

1. Temperature Trend (Line Chart)

2. Humidity Level (Bar Chart)

## Output

- Temperature Trend Graph

- Humidity Bar Chart

- Weather Analysis Report

## Run

```bash
pip install -r requirements.txt
python main.py
```
