#  Data Visualization & Sales Analysis Project

## Project Description

This project demonstrates a complete data analysis workflow using Python, Pandas, and Matplotlib. The sales dataset was loaded, cleaned, analyzed, and visualized using different chart types.

## Steps Performed

1. Loaded sales data from CSV
2. Checked and cleaned data
3. Removed duplicate records
4. Calculated total sales
5. Identified best-selling product
6. Created visualizations

## Charts Created

### Bar Chart

Shows total sales by product category.

### Line Chart

Shows sales trend over time.

### Pie Chart

Shows sales distribution across regions.

## Key Insights

* Total sales revenue calculated successfully
* Best-selling product identified
* Regional sales distribution analyzed

## Technologies Used

* Python
* Pandas
* Matplotlib
