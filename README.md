# 🌦️ Weather Data Analysis using Python

## 📌 Project Overview

This project analyzes weather data from 100 cities using Python. It demonstrates the complete data analysis pipeline, including data loading, cleaning, exploration, visualization, and report generation. The project uses Pandas for data analysis and Matplotlib for data visualization.

---

## 🎯 Objectives

- Load weather data from a CSV file.
- Clean and validate the dataset.
- Perform exploratory data analysis (EDA).
- Calculate key weather statistics.
- Create multiple visualizations.
- Generate a summary report with insights.

---

## 🛠️ Technologies Used

- Python 3
- Pandas
- Matplotlib
- CSV Dataset

## 📊 Dataset Information

The dataset contains weather information for **100 Indian cities** over **10 days**.

### Columns

| Column | Description |
|---------|-------------|
| Date | Observation date |
| City | City name |
| Temperature | Temperature in °C |
| Humidity | Humidity (%) |
| WindSpeed | Wind speed (km/h) |
| Rainfall | Rainfall (mm) |
| Pressure | Atmospheric pressure (hPa) |
| Weather | Weather condition |

## 💡 Key Insights

- Weather data was collected from **100 cities**.
- Sunny weather was observed most frequently.
- Rainfall mainly occurred during Rainy and Thunderstorm conditions.
- Humidity increased during rainy weather.
- Temperature varied across different cities.
- Wind speed remained moderate in most locations.
- Atmospheric pressure stayed relatively stable.
- The dataset was cleaned before analysis.
